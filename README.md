# La Casa de la Hormiga

Sitio web de La Casa de la Hormiga - Live Sessions de música en vivo en Barcelona.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

Generar sitio estático:

```bash
npm run generate
```

El sitio se genera en `.output/public/`

## Environment Variables

Para el formulario de contacto, configurar:

```
WEB3FORMS_ACCESS_KEY=your_key_here
```
